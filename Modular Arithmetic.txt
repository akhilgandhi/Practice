Modular Arithmetic

>   Modular Addition:

    (a + b) mod m = ((a mod m) + (b mod m)) mod m

    For example:
    
    (15 + 17) % 7
    = ((15 % 7) + (17 % 7)) % 7
    = (1 + 3) % 7
    = 4 % 7
    = 4

>   Modular Multiplication:

    (a * b) mod m = ((a mod m) * (b mod m)) mod mod

    For example:

    (12 * 13) % 5
    = ((12 % 5) *  (13 % 5)) % 5
    = (2 * 3) % 5
    = 6 % 5
    = 1
